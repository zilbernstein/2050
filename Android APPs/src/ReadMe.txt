IDE для сборки: Android Studio
	https://developer.android.com/studio?hl=ru


Сторонние библиотеки:
	подключены в виде gradle зависимостей и не трубуют отдельной настройки при сборке.


Для запуска:
	В Android Studio выбрать Tools -> AVD Manager
	Создать новое виртуальное устройство
	использовать реальное устройство или эмулятор с ОС Android 5.0 - 9.0
